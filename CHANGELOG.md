# CHANGELOG

すべての変更はこのファイルに記録されます。

## [1.0.2] - 2026-05-02

### 追加
- **AnkiDec生成時の一括形式指定機能**
    - 「既存小問と別のAnkiDecを作成」画面に、生成する問題の形式を一括で指定できるドロップダウンを追加。
    - 分野別の設定を一時的に上書きし、すべての分野で「正誤」「5肢択一」「穴埋め」などの特定の形式を揃えてAnkiカードを生成することが可能に。

## [1.0.1] - 2026-05-02

### 追加
- **目隠し問題（Image Occlusion）のインテリジェント取込**
    - Ankiのノートタイプから目隠し問題を自動判別する機能を追加。
    - 目隠しが載っていない「綺麗な元画像」を優先的に抽出。
    - 「隠されていた正解の言葉」を画像の知識（要約データ）としてAIに学習させることで、目隠し機能がなくてもAIが重要部位を正確に解説・出題できるよう改善。
- **分野の自動分割（オート・シャーディング）**
    - インポート時、1つの分野（タグ）に21枚以上のカードがある場合、20枚ずつのサブトピック（「分野名 (1)」「分野名 (2)」...）に自動分割して登録する機能を追加。
    - 大量（数百枚規模）の画像があっても、すべての画像が漏れなく説明とテストの対象になるよう改善。

### 修正
- **問題登録機能の正常化**
    - スキップされていた問題登録プロセスを有効化。
    - 取り込んだすべてのカードが「既知の問題」としてデータベースに保存されるよう修正。これにより、既存のカードとAIが生成する新しい問題を組み合わせた効率的な学習が可能に。
- **採点ロジックの改善（Tkinter版）**
    - 未回答（テキストなし・画像添付なし）の設問において、AIが参考画像を回答と誤認して正解にしてしまう不具合を修正。
